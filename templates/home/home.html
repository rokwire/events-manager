Introduction

Events Manager is the administrative interface to events in the Rokwire Platform. Through this web application, users can manage existing events coming from calendars and or create and manage user events.

Main features of Events Manager are as follows:

User Events

  • View events (published and pending)
  • Manage events (create, view, edit, delete, and publish)
  • Send notifications

Calendar Events

  • View event sources
  • View events (published and pending)
  • Manage events (publish, make pending, and delete)
  • Manage calendars (add and approve)
  • Schedule daily downloads
